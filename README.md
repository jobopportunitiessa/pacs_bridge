## üìñ Overview

pacs_bridge is a software solution that links Hospital Information Systems (HIS) and Radiology Information Systems (RIS) to a Picture Archiving and Communication System (PACS). It serves as a bridge for HL7 v2.x and FHIR R4 data formats, facilitating smooth healthcare integration. 

### üåü Key Features
- **HL7 and FHIR Support**: Connects HIS and RIS with modern data formats for better information flow.
- **DICOM Compatibility**: Supports DICOM Modality Worklist (MWL) and Modality Performed Procedure Step (MPPS) for streamlined medical imaging processing.
- **User-Friendly Interface**: Simple navigation for all users, ensuring easy setup and operation without technical expertise.
